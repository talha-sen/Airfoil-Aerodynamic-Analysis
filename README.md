# Airfoil Aerodynamic Analysis – NACA 2412

This project presents the aerodynamic analysis of the NACA 2412 airfoil using XFLR5.

## Analysis Performed
- Lift coefficient (CL) vs Angle of Attack (α)
- Drag coefficient (CD) vs CL (Drag Polar)
- Pitching moment coefficient (Cm) vs Angle of Attack (α)

## Key Results
- CLmax ≈ 1.3
- Stall angle ≈ 15


## Methodology
- Airfoil: NACA 2412  
- Reynolds Number: 500,000  
- Angle of Attack Range: -5° to 15°  
- Software: XFLR5
